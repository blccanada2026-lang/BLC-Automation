Bulk Staff Import

How to use:
1. Open the STG_STAFF_IMPORT sheet in the BLC Nexus spreadsheet.
2. Fill in one row per staff member. Required: person_code, name, email, role, pay_currency, pay_design, pay_qc, effective_from.
3. Banking fields (IFSC, SWIFT, etc.) are optional — can be added later.
4. Click Run Import below. The system writes import_status back to each row.
5. Already-imported rows (IMPORTED) are automatically skipped — safe to re-run.
